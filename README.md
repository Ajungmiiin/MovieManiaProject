# MOVIE MANIA PROJECT
TMDB API와 NextJs 를 이용해 제작한 영화 리뷰 작성 사이트입니다.

## 주요기능
- 모바일 **반응형** 웹 사이트
- **회원가입 & 로그인**
- 기본적인 **CRUD** ( 북마크 & 리뷰 ... )
- **Toast UI** 를 활용하여 사용자에게 **즉각적인 피드백** 제공
